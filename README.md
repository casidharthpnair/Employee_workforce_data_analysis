Employee Workforce Data Analysis
Project Overview
This project performs an end-to-end analysis of employee workforce data using Python. The objective is to clean, transform, analyze, and visualize employee-related information to uncover insights about workforce demographics, salaries, performance, departments, and employee status.
Problem Statement
Organizations rely on workforce data to make informed decisions regarding employee performance, compensation, and resource allocation. This project demonstrates how data analytics can be used to identify trends, improve workforce management, and support data-driven decision-making.

Key Insights
Salary Analysis
•	Most employees earn between ₹80,000 and ₹100,000.
•	Average Salary: ₹85,164.30
•	Highest Salary: ₹119,971.65
•	Lowest Salary: ₹50,047.32
Department Analysis
•	DevOps and Sales departments have the highest employee count.
•	Cloud Tech has the lowest employee representation.
Age Analysis
•	Most employees are between 25 and 40 years old.
•	Workforce distribution is slightly right-skewed, indicating a larger concentration of employees in their early 30s.
